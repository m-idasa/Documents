# حراج بازار فیزیکی

## 📌 آدرس سرویس

```
POST http://{BaseUrl}/Energy/Auction
```

مثال:

```bash
curl --location 'https://{BaseUrl}/Energy/Auction' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {token}' \
--data '{
    "Idn": 0
}'
```

## 🧾 نام سرویس

`Auction`

---

## 🎯 توضیحات

سرویس **Auction** فهرست پیغام های حراج بازار فیزیکی را ارائه می‌کند.

---

## درخواست

پارامترهای ورودی در قالب JSON ارسال می‌گردد.

### 📥 پارامترهای ورودی

Body (JSON):

| نام فیلد  | نوع داده  | توضیحات |
|-----------|-----------|---------|
| Idn    | long   | شماره آخرین رکورد دریافتی در فراخوانی قبلی - مرتبه اول 0 استفاده شود |

## پاسخ

پاسخ در قالب JSON برمی‌گردد.

### 📤 ساختار خروجی:

| نام ستون | توضیحات |
|---|---|
| Idn | شماره رکورد (کلید یکتا) |
| Message | متن پیام |
| MessageType | نوع پیام |
| Deven | تاریخ |
| Heven | ساعت |
| AuctionID | شماره حراج |

### مقادیر LoadType
- 0: بار پایه
- 1: کم‌باری
- 2: میان‌باری
- 3: پرباری

---

## 📦 نمونه ورودی 

```json
{
  "Idn": 0
}
```

## 📄 نمونه خروجی

```json
{
  "IsSuccess": true,
  "Data":
    [
      {
        "Idn": 1,
        "Message": "پیام نمونه",
        "MessageType": "Info",
        "Deven": 20250817,
        "Heven": 103000,
        "AuctionID": 12345
      }
    ],
  "Msg": null
}
```

---

## ✍️ نکات تکمیلی
- حتماً باید در هدر درخواست، **توکن JWT معتبر** ارسال شود.
- فقط کاربرانی که نقش `Auction` دارند می‌توانند این سرویس را فراخوانی کنند.
